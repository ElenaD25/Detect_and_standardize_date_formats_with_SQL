# Detect_and_standardize_date_formats_with_SQL
This project provides SQL scripts to detect and standardize various date formats found in improperly designed columns (e.g., VARCHAR instead of DATE). If you've ever inherited a table where dates were stored as text, this tool will help you bring order to the chaos 🤯

This is V1 — it just detects and shows the “before vs after” to the client… but updates absolutely nothing 😅 (yes, it’s basically all talk, no action)
